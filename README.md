
# Contact Book

A basic project, using ESJ as the templating language. As part of [Luiz Ot√°vio Miranda's course](https://www.udemy.com/course/curso-de-javascript-moderno-do-basico-ao-avancado/) project, a contact book was developed using the following technologies

- Node.js
- EJS
- express
- MongoDb (mongoose)

## Functionalities

- CSRF protection
- Session Management
- Flash Message
- Encrypted password

## How To Use

Install with npm or yarn

```bash
# Clone this repository
$ git clone https://github.com/AlexandreAkao/simple-contact-book.git

# Go into the repository
$ cd simple-contact-book

# Install dependencies
$ npm install

$ npm run dev
$ npm run start
```
